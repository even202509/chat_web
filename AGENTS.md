# Project:Vibe Chat

你是一位经验丰富的全栈开发工程师，负责对现有项目进行完善与维护，根据用户的需求与你的经验修改项目文件,数据库由其他人负责，

### 数据库
#### users

- id            | integer                  |          | not null | generated by default as identity
- username      | character varying(80)    |          | not null |
- password_hash | character varying(256)   |          | not null |
- created_at    | timestamp with time zone |          |          | CURRENT_TIMESTAMP
- last_login    | timestamp with time zone |          |          |
- is_active     | boolean                  |          |          | true

#### friends
- id          | integer                  |          | not null | nextval('friends_id_seq'::regclass)
- outgoing_id | integer                  |          | not null |
- incoming_id | integer                  |          | not null |
- status      | character varying(20)    |          | not null | 'pending'::character varying
- created_at  | timestamp with time zone |          |          | CURRENT_TIMESTAMP

#### messages
```
CREATE TABLE messages (
    id BIGSERIAL NOT NULL,
    chat_id VARCHAR(50) NOT NULL,        -- 私聊："user1:user2"，群聊："group:123"
    chat_type VARCHAR(20) NOT NULL,      -- 'private' 或 'group'
    sender_id INT NOT NULL,
    msg_type VARCHAR(20) DEFAULT 'text', -- 'text', 'image', 'file', 'system'
    content TEXT,
    reply_to_id BIGINT,                   -- 回复的消息ID
    created_at TIMESTAMP DEFAULT NOW(),
    
    -- 全局删除（管理员/群主）
    deleted_at TIMESTAMP,
    deleted_by INT
);
```
#### user_message_status
```
CREATE TABLE user_message_status (
    id BIGSERIAL PRIMARY KEY,
    user_id INT NOT NULL,
    message_id BIGINT NOT NULL,
    chat_id VARCHAR(50) NOT NULL,         -- 冗余字段，方便查询
    chat_type VARCHAR(20) NOT NULL,
    
    -- 状态标记
    is_read BOOLEAN DEFAULT FALSE,
    read_at TIMESTAMP,
    is_deleted BOOLEAN DEFAULT FALSE,      -- 用户自己删除
    deleted_at TIMESTAMP,
    
    -- 群聊特殊字段
    mention_type VARCHAR(20),              -- 'none', 'all', 'someone'
    is_mentioned BOOLEAN DEFAULT FALSE,
    
    created_at TIMESTAMP DEFAULT NOW(),
    
    -- 唯一约束
    UNIQUE(user_id, message_id)
);
```

#### groups
- id         | integer                     |          | not null | nextval('groups_id_seq'::regclass)
- group_name | character varying(100)      |          | not null |
- creator_id | integer                     |          | not null |
- created_at | timestamp without time zone |          |          | CURRENT_TIMESTAMP

#### chat_members
```
CREATE TABLE group_members (
    id BIGSERIAL PRIMARY KEY,
    chat_id VARCHAR(50) NOT NULL,
    user_id INT NOT NULL,
    role VARCHAR(20) DEFAULT 'member',     -- 'owner', 'admin', 'member'
    
    -- 用户在该聊天的个性化设置
    alias VARCHAR(100),                    -- 群昵称
    is_muted BOOLEAN DEFAULT FALSE,        -- 免打扰
    mute_until TIMESTAMP,                  -- 免打扰截止时间
    
    joined_at TIMESTAMP DEFAULT NOW()
);
```

## 开发规范
1. 代码质量优先
2. 在完成一个业务实现后进行记录
3. 代码复用性尽量高，但不要违背第一条原则
3. 不要添加额外的技术栈
5. 如果现有数据库无法满足需求，请向用户说明你想添加或修改的表
6. 可以对项目未来的升级提供建议
